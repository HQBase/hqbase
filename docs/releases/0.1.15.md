# HQBase Community 0.1.15

- Fresh Deploy-to-Cloudflare installations now deploy the verified signed stable artifact and carry
  its immutable Community release identity.
- Community updates now compare the stable release with the active production Worker version rather
  than the cloned repository version.
- Pro upgrades verify the active Community Worker against its exact versioned signed manifest and
  give unsigned legacy deployments a specific Community-update recovery action.
- The update banner is shown only when the advertised release is directly compatible.
