# HQBase Community 0.1.4

- Separates the owner account email from workspace mailboxes, so sign-in and recovery can use any
  valid email address.
- Keeps every Community mailbox shared and restricted to the selected Cloudflare workspace domain.
- Adds an explicit, default-off option to create an owner-named shared mailbox during setup.
- Aligns the mail header controls and uses the HQBase brand mark in the application shell.
- Improves setup reliability with browser-safe Cloudflare authorization, clearer progress, automatic
  per-install auth secrets, correct Worker identity propagation, and API routing ahead of SPA assets.
- Supports direct updates from Community 0.1.3.
